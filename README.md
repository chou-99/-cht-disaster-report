# 災情即時通報平台

此專案提供一個簡易的災情通報網站原型，讓民眾可以快速填寫災害資訊並上傳現場照片，資料會即時呈現在頁面上，模擬地方政府或社區在災害發生時的資訊整合平台。

## 功能特色

- 📣 重要訊息公告區，提供氣象特報與聯絡資訊。
- 📝 線上通報表單，包含地點、類型、描述與照片上傳欄位。
- 🖼️ 即時照片預覽，協助掌握現場狀況。
- 🔍 災害類型篩選，可依需求快速找到相關通報。
- 📍 避難所與物資資訊，提供避難與救援參考。
- 💾 透過瀏覽器 `localStorage` 保存資料（僅在本機瀏覽器中暫存）。

## 使用方式

1. 直接在瀏覽器中開啟 `index.html` 即可使用。
2. 填寫表單後送出，通報資料會立即出現在「最新災情通報」區域。
3. 若有上傳照片，會在通報卡片中顯示。
4. 下拉選單可依災害類型篩選通報內容。

> ⚠️ 此為前端原型，僅示範使用體驗與資訊呈現。實際上線時需搭配後端服務與權限驗證確保資料安全。

## 開發

此專案使用原生 HTML、CSS 與 JavaScript 實作，無需額外建置環境。

```bash
# 以靜態伺服器在本機預覽（舉例使用 Python）
python -m http.server 8000
```

在瀏覽器開啟 `http://localhost:8000` 即可預覽頁面。
